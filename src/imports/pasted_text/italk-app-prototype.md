=== 프롬프트 시작 ===

부모와 자녀의 실제 대화를 분석해 더 나은 소통법을 코칭하는 모바일 앱 **"아이톡(iTalk)"**의 고해상도 인터랙티브 프로토타입을 만들어줘. 영어교육 앱 **링글(Ringle)**의 `기록 → AI 분석 → 피드백 리포트 → 복습·연습` 학습 흐름을, 영어 교정 대신 **부모-자녀 소통 코칭**에 그대로 적용한 컨셉이야. 톤은 밝고 다정하며, 육아에 지친 부모를 따뜻하게 응원하는 느낌.

## 1. 기본 설정
- 플랫폼: **iOS 모바일 앱**, 아이폰 프레임 **390×844** (iPhone 15 기준), 세로 화면
- 실제로 화면 전환이 동작하는 하이파이 프로토타입으로 제작
- 모든 UI 텍스트는 **한국어**
- 하단 고정 **탭바 네비게이션** + 중앙 플로팅 기록 버튼

## 2. 컬러 시스템 (캡쳐에서 추출)
| 역할 | 이름 | HEX |
|---|---|---|
| **Primary / 메인** | 스카이 블루 | `#2BC4F0` |
| Primary Dark (눌림·강조·텍스트) | 딥 블루 | `#15A6D8` |
| Primary Light (틴트 배경) | 페일 블루 | `#E6F7FD` |
| **Point / 포인트** | 크림 옐로우(카드) | `#FBEFA8` |
| Point Strong (로고·하이라이트) | 버터 옐로우 | `#FBE07A` |
| 텍스트 900 (제목) | 잉크 | `#1B1B1F` |
| 텍스트 500 (본문·보조) | 그레이 | `#6B7280` |
| 서피스 | 화이트 | `#FFFFFF` |
| 배경 | 라이트 | `#F6FBFE` |
| 감정·상태 - 긍정 | 그린 | `#34C759` |
| 감정·상태 - 주의/번아웃 | 코랄 | `#FF7A6B` |

- 주요 버튼·브랜드·강조 = 스카이블루, 응원/코칭 카드 = 크림 옐로우, 본문 면 = 화이트.
- 그라데이션 허용: 스카이블루 → 살짝 더 밝은 시안(`#2BC4F0 → #5FD6F7`)을 히어로/버튼에 은은하게.

## 3. 타이포그래피
- 한글 서체 **Pretendard** (없으면 system sans, 둥근 느낌 우선)
- 화면 타이틀 26 Bold / 섹션 타이틀 20 SemiBold / 본문 15~16 Regular~Medium / 캡션 12~13 / 숫자 강조 30~40 Bold
- 행간 넉넉하게(1.4~1.5), 자간 0

## 4. 모양 · 깊이
- 카드 라운드 **20~24**, 버튼 **16**, 칩/뱃지 **pill(999)**, 입력창 14
- 그림자: 낮고 부드럽게 `0 8 24 rgba(20,40,60,0.08)`
- 좌우 여백 20, 카드 간격 14~16
- 아이콘: 둥근 라인 아이콘(스트로크 2), 일러스트는 통통한 3D·플랫 혼합
- 마스코트: 웃는 둥근 캐릭터(밝은 블루 바탕에 크림색, 위 캡쳐의 'ö' 눈웃음 느낌 차용). 빈 화면·로딩·완료 등에 등장

## 5. 정보구조 (하단 탭 5개)
`홈` · `코칭` · `＋ 기록(중앙 플로팅)` · `리포트` · `마이`
- 중앙 ＋ 버튼은 스카이블루 원형 FAB, 탭하면 **음성 녹음 / 텍스트 입력 / 대화 캡쳐 업로드** 3갈래로 펼쳐지는 라디얼 메뉴

## 6. 화면별 상세

**(1) 스플래시**
- 스카이블루 풀배경, 중앙에 아이톡 로고(크림 옐로우). 로고가 통통 튀며 등장.

**(2) 온보딩 3장 (스와이프)**
- ① "우리 아이와의 대화, 그대로 들려주세요" ② "AI가 소통 방식을 분석해 드려요" ③ "오늘부터 더 다정한 대화 연습" — 각 장 둥근 일러스트 + 하단 인디케이터, [시작하기] 버튼.

**(3) 로그인 / 회원가입**
- 카카오·애플 간편 로그인 버튼, 이메일 로그인. 상단 마스코트.

**(4) 자녀 프로필 설정**
- 자녀 이름, 연령대(영유아/유아/초등 저학년/고학년 칩 선택), 관계(엄마/아빠/기타), 프로필 이모지 선택. 여러 자녀 추가 가능.

**(5) 홈 / 대시보드** *(레이아웃은 위 Bave 캡쳐 톤을 차용)*
- 상단: 로고 + 알림🔔 + 설정⚙
- 인사 영역: **"안녕하세요, 지영님!"** + "오늘도 우리 아이와 좋은 대화 나눠요."
- **오늘의 응원** 카드(크림): "어제 아이에게 '고마워'라고 3번이나 말했어요. 따뜻한 하루였어요."
- **오늘의 코칭** 카드(크림): "공감 표현이 조금 부족했어요. 5분 코칭을 시작해볼까요?" → [확인 →]
- **이번 주 기록** 섹션: 가로 3분할 통계 카드 — `분석 12회 / 이번 주`, `소통 점수 78점 / 평균 +5`, `공감지수 좋음`
- 하단 **빠른 기록** 블루 카드: 마이크/카메라 아이콘 + "대화를 들려주세요. AI가 분석해 코칭해드릴게요." + 큰 ＋ 버튼

**(6) 기록(분석 요청) 화면**
- 상단 탭: [음성 녹음] / [텍스트 입력] / [캡쳐 업로드]
- 음성 녹음: 중앙 큰 마이크 버튼, 둘레에 **실시간 음성 파형(waveform)**과 ripple 펄스, 녹음 시간 카운터, [일시정지][완료]
- 텍스트: 화자 구분(부모/아이) 말풍선 입력
- 업로드: 카톡 등 대화 캡쳐 이미지 첨부 영역

**(7) AI 분석 로딩**
- 마스코트가 헤드폰 끼고 듣는 일러스트, "대화를 분석하고 있어요…" + 단계별 체크 진행(① 음성 인식 ② 감정 톤 분석 ③ 코칭 생성), 맥동하는 점 로딩.

**(8) 분석 리포트 (핵심 화면)** *(링글의 CAF 피드백 리포트 차용)*
- 상단: **소통 점수 원형 게이지 82점**(0→82 채워지는 애니메이션 + 숫자 카운트업), "지난 대화보다 +6점"
- **감정 톤 곡선**: 대화 흐름에 따른 부모/아이 감정 라인 차트(그려지는 애니메이션)
- **소통 분석 막대**: 공감 표현 / 열린 질문 / 칭찬·격려 / 지시·명령 비율 (자라나는 막대)
- **잘한 점** (그린 카드): "아이의 감정을 먼저 알아줬어요 — '많이 속상했구나'"
- **이렇게 말해보세요** (블루 카드, 링글의 교정 제안 차용): 실제 발화 → 추천 표현 비교 *"빨리 해!" → "5분 뒤에 시작할까? 네가 정해볼래?"*
- 하단 [이 상황 코칭 받기] [리포트 저장]

**(9) 코칭 라이브러리 (레슨)** *(링글 교재 목록 차용)*
- 상황별 미니 레슨 카드 그리드: `떼쓰고 울 때`, `칭찬·격려하는 법`, `훈육이 필요할 때`, `화난 아이 공감하기`, `형제 다툼 중재`, `잠자기 전 대화`, `스마트폰 규칙 정하기`. 각 카드 썸네일·소요시간·난이도.

**(10) 레슨 상세 / AI 롤플레이 연습** *(링글 AI 튜터 차용)*
- 상단 핵심 포인트 3가지 + 예시 대화. 하단 [AI와 연습하기] → 채팅형 롤플레이: AI가 '아이' 역할, 부모가 답하면 즉석 피드백("공감 먼저! 👍")과 점수.

**(11) 리포트 모음 / 성장 추이**
- 주간·월간 토글, **소통 점수 추이 라인 그래프**(왼→오 그려짐), 항목별 변화(공감↑·지시↓), 분석 히스토리 리스트(날짜·점수·요약).

**(12) 마이페이지**
- 프로필·자녀 관리, 누적 분석·연속 기록(스트릭), 알림 설정, 구독(아이톡 프리미엄), 고객센터.

## 7. 핵심 컴포넌트
- 하단 탭바(아이콘+라벨, 활성 탭 스카이블루) + 중앙 블루 FAB
- 카드(크림 응원형 / 화이트 정보형 / 블루 액션형)
- 소통 점수 원형 게이지, 막대·도넛·라인 차트
- 칩/필터, pill 버튼, 토글, 말풍선(부모=블루·아이=크림)
- 녹음 위젯(파형+타이머), 빈 상태 일러스트(마스코트)

## 8. 애니메이션 · 인터랙션 *(반드시 풍부하게)*
- **스플래시**: 로고 스케일 인 + 통통 bounce, 살짝 페이드
- **온보딩**: 슬라이드 전환 시 일러스트 패럴랙스, 둥둥 떠다니는 floating loop
- **홈 진입**: 카드들이 아래에서 위로 **순차(stagger) fade-up**
- **소통 점수 게이지**: 0→점수까지 원호가 채워지고 숫자 **카운트업**
- **차트**: 막대는 아래에서 자라나고(grow), 라인은 왼→오 **draw**, 도넛은 회전하며 채워짐
- **녹음**: 마이크 둘레 ripple 펄스 + 실시간 waveform, 녹음 시작/종료 시 버튼 morph
- **AI 분석 로딩**: 단계 체크가 하나씩 ✓로 바뀌고 점 로딩 맥동
- **버튼**: 누르면 scale 0.96 + 살짝 그림자 축소(햅틱 느낌)
- **탭 전환**: 콘텐츠 cross-fade, 활성 탭 아이콘 bounce
- **중앙 FAB**: 탭 시 3개 옵션이 **라디얼로 펼쳐짐**(스프링)
- **카드 → 상세**: shared element transition(카드가 확장되며 리포트로)
- **코칭 카드**: 좋아요 시 하트 pop, 완료 시 컨페티 살짝
- **Pull-to-refresh**: 마스코트가 통통 튀는 커스텀 인디케이터
- 전체 전환은 부드러운 ease-out, 240~320ms 권장

## 9. 더미 데이터 (그대로 사용)
- 사용자: 지영(엄마) · 자녀: 채이(5세)
- 이번 주 소통 점수 78, 최근 리포트 82(+6)
- 응원 문구: "어제 채이가 세 번이나 웃었어요. 엄마의 다정한 목소리 덕분이에요."
- 교정 예시: "빨리 좀 해!" → "5분 뒤에 시작할까? 네가 정해볼래?"

전체적으로 밝은 스카이블루와 크림 옐로우가 어우러진, 둥글고 다정하고 깔끔한 모바일 앱으로 완성해줘.

### 링글에서 차용한 UI/UX 패턴 (참고)
- **학습 루프 구조**: 링글의 `예습 → 레슨 → 복습 → 피드백 리포트` → 아이톡의 `대화 기록 → AI 분석 → 리포트 → 코칭·연습`
- **CAF 피드백 리포트**: 링글이 문법/어휘/유창성을 분석·교정 → 아이톡은 공감/질문/칭찬/지시 비율·감정 톤을 분석하고 "이렇게 말해보세요"로 교정 제안
- **교재 라이브러리** → 상황별 코칭 미니 레슨
- **AI 튜터 실시간 연습** → AI 롤플레이(아이 역할) 대화 연습
- **개인화 대시보드 / 성장 추이 그래프** → 홈 대시보드·주간 리포트
