# 아이톡(iTalk) · Figma Make 프롬프트

> **사용법** — 아래 `=== 프롬프트 시작 ===` 부터 `=== 프롬프트 끝 ===` 까지 전체를 복사해 Figma Make 입력창에 붙여넣으세요. 화면별로 나눠 생성하고 싶으면 「6. 화면별 상세」의 해당 섹션만 떼어 추가 프롬프트로 사용하면 됩니다.
>
> **메인 색상 출처** — 업로드하신 *Bave* 캡쳐에서 추출했습니다. 배경의 밝은 스카이블루를 Primary, 카드의 크림 옐로우를 Point 컬러로 잡았습니다. (눈으로 정밀 매칭한 값이라 피그마에서 1~2 단계 미세조정 가능)
>
> **v2 추가** — ① 동화 읽어주기 + 끝말잇기 「놀이」 콘텐츠, ② 열린/폐쇄형 질문 등 언어교육학 근거 코칭을 화면에 명시(섹션 10).

---

=== 프롬프트 시작 ===

부모와 자녀의 실제 대화를 분석해 더 나은 소통법을 코칭하고, 동화 읽어주기·끝말잇기로 함께 놀며 언어를 키우는 모바일 앱 **"아이톡(iTalk)"**의 고해상도 인터랙티브 프로토타입을 만들어줘. 영어교육 앱 **링글(Ringle)**의 `기록 → AI 분석 → 피드백 리포트 → 복습·연습` 학습 흐름을, 영어 교정 대신 **부모-자녀 소통 코칭**에 적용한 컨셉이야. 코칭 내용은 **언어교육학(대화식 책읽기, 열린 질문 등)** 에 근거해. 톤은 밝고 다정하며, 육아에 지친 부모를 따뜻하게 응원하는 느낌.

## 1. 기본 설정
- 플랫폼: **iOS 모바일 앱**, 아이폰 프레임 **390×844** (iPhone 15 기준), 세로 화면
- 실제로 화면 전환이 동작하는 하이파이 프로토타입으로 제작
- 모든 UI 텍스트는 **한국어**
- 하단 고정 **탭바 네비게이션** + 중앙 플로팅 기록 버튼

## 2. 컬러 시스템 (캡쳐에서 추출)
| 역할 | 이름 | HEX |
|---|---|---|
| **Primary / 메인** | 스카이 블루 | `#2BC4F0` |
| Primary Dark (눌림·강조·텍스트) | 딥 블루 | `#15A6D8` |
| Primary Light (틴트 배경) | 페일 블루 | `#E6F7FD` |
| **Point / 포인트** | 크림 옐로우(카드) | `#FBEFA8` |
| Point Strong (로고·하이라이트) | 버터 옐로우 | `#FBE07A` |
| 텍스트 900 (제목) | 잉크 | `#1B1B1F` |
| 텍스트 500 (본문·보조) | 그레이 | `#6B7280` |
| 서피스 | 화이트 | `#FFFFFF` |
| 배경 | 라이트 | `#F6FBFE` |
| 감정·상태 - 긍정 | 그린 | `#34C759` |
| 감정·상태 - 주의/번아웃 | 코랄 | `#FF7A6B` |

- 주요 버튼·브랜드·강조 = 스카이블루, 응원/코칭 카드 = 크림 옐로우, 본문 면 = 화이트.
- 그라데이션 허용: 스카이블루 → 살짝 더 밝은 시안(`#2BC4F0 → #5FD6F7`)을 히어로/버튼에 은은하게.

## 3. 타이포그래피
- 한글 서체 **Pretendard** (없으면 system sans, 둥근 느낌 우선)
- 화면 타이틀 26 Bold / 섹션 타이틀 20 SemiBold / 본문 15~16 Regular~Medium / 캡션 12~13 / 숫자 강조 30~40 Bold
- 행간 넉넉하게(1.4~1.5), 자간 0

## 4. 모양 · 깊이
- 카드 라운드 **20~24**, 버튼 **16**, 칩/뱃지 **pill(999)**, 입력창 14
- 그림자: 낮고 부드럽게 `0 8 24 rgba(20,40,60,0.08)`
- 좌우 여백 20, 카드 간격 14~16
- 아이콘: 둥근 라인 아이콘(스트로크 2), 일러스트는 통통한 3D·플랫 혼합
- 마스코트: 웃는 둥근 캐릭터(밝은 블루 바탕에 크림색, 위 캡쳐의 'ö' 눈웃음 느낌 차용). 빈 화면·로딩·완료·놀이에 등장

## 5. 정보구조 (하단 탭 5개)
`홈` · `코칭` · `＋ 기록(중앙 플로팅)` · `놀이` · `마이`
- 중앙 ＋ 버튼은 스카이블루 원형 FAB, 탭하면 **음성 녹음 / 텍스트 입력 / 대화 캡쳐 업로드** 3갈래로 펼쳐지는 라디얼 메뉴
- **놀이** 탭 = 부모·자녀가 함께하는 콘텐츠 허브(동화 읽어주기·끝말잇기)
- **리포트(성장 추이)** 는 홈의 소통 점수 카드를 탭하면 진입

## 6. 화면별 상세

**(1) 스플래시**
- 스카이블루 풀배경, 중앙에 아이톡 로고(크림 옐로우). 로고가 통통 튀며 등장.

**(2) 온보딩 3장 (스와이프)**
- ① "우리 아이와의 대화, 그대로 들려주세요" ② "AI가 소통 방식을 분석해 드려요" ③ "동화·끝말잇기로 함께 놀며 말이 자라요" — 각 장 둥근 일러스트 + 하단 인디케이터, [시작하기] 버튼.

**(3) 로그인 / 회원가입**
- 카카오·애플 간편 로그인 버튼, 이메일 로그인. 상단 마스코트.

**(4) 자녀 프로필 설정**
- 자녀 이름, 연령대(영유아/유아/초등 저학년/고학년 칩 선택), 관계(엄마/아빠/기타), 프로필 이모지 선택. 여러 자녀 추가 가능. (연령대는 놀이 난이도·동화 추천에 사용)

**(5) 홈 / 대시보드** *(레이아웃은 위 Bave 캡쳐 톤을 차용)*
- 상단: 로고 + 알림🔔 + 설정⚙
- 인사 영역: **"안녕하세요, 지영님!"** + "오늘도 우리 아이와 좋은 대화 나눠요."
- **오늘의 응원** 카드(크림): "어제 아이에게 '고마워'라고 3번이나 말했어요. 따뜻한 하루였어요."
- **오늘의 코칭** 카드(크림): "어제는 닫힌 질문이 많았어요. 오늘은 '왜', '어떻게'로 물어볼까요?" → [코칭 보기 →]
- **이번 주 기록** 섹션: 가로 3분할 통계 카드 — `분석 12회 / 이번 주`, `소통 점수 78점 / 평균 +5`, `열린 질문 64%`. (점수 카드 탭 → 리포트 모음)
- **함께 놀기** 추천 카드(블루): "오늘의 동화 「용감한 토끼 코코」 · 읽고 끝말잇기까지" → [놀이 시작]
- 하단 **빠른 기록** 블루 카드: 마이크/카메라 아이콘 + "대화를 들려주세요. AI가 분석해 코칭해드릴게요." + 큰 ＋ 버튼

**(6) 기록(분석 요청) 화면**
- 상단 탭: [음성 녹음] / [텍스트 입력] / [캡쳐 업로드]
- 음성 녹음: 중앙 큰 마이크 버튼, 둘레에 **실시간 음성 파형(waveform)**과 ripple 펄스, 녹음 시간 카운터, [일시정지][완료]
- 텍스트: 화자 구분(부모/아이) 말풍선 입력
- 업로드: 카톡 등 대화 캡쳐 이미지 첨부 영역

**(7) AI 분석 로딩**
- 마스코트가 헤드폰 끼고 듣는 일러스트, "대화를 분석하고 있어요…" + 단계별 체크 진행(① 음성 인식 ② 감정 톤 분석 ③ 질문·표현 분석 ④ 코칭 생성), 맥동하는 점 로딩.

**(8) 분석 리포트 (핵심 화면)** *(링글의 CAF 피드백 리포트 차용)*
- 상단: **소통 점수 원형 게이지 82점**(0→82 채워지는 애니메이션 + 숫자 카운트업), "지난 대화보다 +6점"
- **감정 톤 곡선**: 대화 흐름에 따른 부모/아이 감정 라인 차트(그려지는 애니메이션)
- **소통 분석 막대**: 열린 질문 / 폐쇄형 질문 / 공감 표현 / 칭찬·격려 / 지시·명령 비율 (자라나는 막대). 열린 질문 비율을 가장 강조.
- **잘한 점** (그린 카드): "아이의 감정을 먼저 알아줬어요 — '많이 속상했구나'"
- **이렇게 말해보세요** (블루 카드): 실제 발화 → 추천 표현을 Before→After로, 각 항목에 〔원리 태그〕 + 한 줄 근거 표시
  - 〔열린 질문〕 "재밌었어?" *(예/아니오로 끝남)* → **"오늘 뭐가 제일 재밌었어?"** — *열린 질문은 더 긴 말과 새 어휘를 끌어내요.*
  - 〔선택 질문〕 "빨리 해!" → **"5분 뒤에 할까, 지금 할까?"** — *명령 대신 선택지를 주면 스스로 정하는 힘이 자라요.*
  - 〔확장·되돌려주기〕 아이 "멍멍 가" → **"응, 강아지가 집으로 가는구나"** — *바르고 길게 되돌려주면 문장 모델이 돼요.*
- 하단 [이 상황 코칭 받기] [리포트 저장]

**(9) 코칭 라이브러리 (레슨)** *(링글 교재 목록 차용)*
- 상단 카테고리 탭: `대화·언어` / `감정·훈육` / `놀이로 배우기`
- `대화·언어` 레슨(언어교육학 근거): `열린 질문으로 대화 늘리기`, `대화식 책읽기 PEER 4단계`, `CROWD 질문 5종`, `확장·되돌려주기(recast)`, `5초만 기다려주기`
- `감정·훈육`: `떼쓰고 울 때`, `칭찬·격려하는 법`, `훈육이 필요할 때`, `화난 아이 공감하기`, `형제 다툼 중재`
- `놀이로 배우기`: `동화로 어휘 늘리기`, `끝말잇기로 소리 익히기`
- 각 카드 썸네일·소요시간·난이도. 각 레슨 상세 첫 화면에 **핵심 원리 1줄 요약 + 예시 대화**.

**(10) 레슨 상세 / AI 롤플레이 연습** *(링글 AI 튜터 차용)*
- 상단 핵심 포인트 3가지 + 예시 대화. 하단 [AI와 연습하기] → 채팅형 롤플레이: AI가 '아이' 역할, 부모가 답하면 즉석 피드백(예: "방금은 닫힌 질문이었어요. '왜?'를 붙여볼까요? 👍")과 점수.

**(11) 놀이(함께놀기) 허브** — 하단 탭 진입
- 인사 + 자녀 연령 맞춤 추천. 큰 카드 2개: 「오늘의 동화 읽어주기」, 「끝말잇기」.
- 아래 **추천 동화** 가로 스크롤(연령·주제별), **지난 놀이 기록**(읽은 동화·끝말잇기 라운드·새 낱말 수).

**(12) 동화 읽어주기** *(대화식 책읽기 PEER·CROWD 적용)*
- 그림책 일러스트 + 문장, 재생/일시정지·속도·자동 넘김. AI가 또박또박 읽어주고, **읽는 단어가 하이라이트**로 따라감.
- 페이지마다 하단에 **"함께 이야기해보세요" 코칭 말풍선**(크림 카드)이 떠서 부모에게 질문 예시 제공 — CROWD 유형 태그와 함께:
  - 〔완성형〕 "토끼는 깡충깡충 ___로 갔어요. 뭐라고 했을까?"
  - 〔회상〕 "방금 토끼가 누굴 만났지?"
  - 〔열린질문〕 "토끼는 왜 무서웠을까? 어떻게 생각해?"
  - 〔육하원칙〕 "여기서 누가 제일 용감했어?"
  - 〔연결〕 "채이도 무서웠던 적 있어? 언제였어?"
- 우측 작은 **PEER 가이드** 토글: 질문하기 → 아이 답 인정 → 한 마디 더해 확장 → 새 낱말 반복.
- 다 읽으면 「오늘 만난 낱말」 정리 카드 + [끝말잇기로 이어가기] 버튼.

**(13) 끝말잇기 게임** *(음운인식·어휘 확장)*
- 말풍선 체인 UI(부모=블루·아이=크림·AI=화이트), 직전 낱말의 **끝 글자 강조**, 누구 차례인지 표시.
- 연령별 난이도(받침 없는 쉬운 낱말 ↔ 도전), **그림 힌트**, 아이가 막히면 AI가 쉬운 낱말로 받아주고 힌트 제공.
- 하단 **음운인식 코칭 팁**(크림 카드): "끝소리에 집중해요 — '사과'의 끝소리는 '과'예요!"
- 새 낱말은 자동으로 **어휘 노트**에 저장. 라운드 끝 요약(이어간 낱말 수·새 어휘·칭찬 스티커).

**(14) 리포트 모음 / 성장 추이**
- 주간·월간 토글, **소통 점수 추이 라인 그래프**(왼→오 그려짐), 항목별 변화(열린 질문↑·지시↓), 분석 히스토리 리스트(날짜·점수·요약). 놀이 활동(읽은 동화·끝말잇기·새 어휘)도 함께 누적 표시.

**(15) 마이페이지**
- 프로필·자녀 관리, 누적 분석·연속 기록(스트릭)·모은 어휘 수, 알림 설정, 구독(아이톡 프리미엄), 고객센터.

## 7. 핵심 컴포넌트
- 하단 탭바(아이콘+라벨, 활성 탭 스카이블루) + 중앙 블루 FAB
- 카드(크림 응원·코칭형 / 화이트 정보형 / 블루 액션형)
- 소통 점수 원형 게이지, 막대·도넛·라인 차트
- 칩/필터, pill 버튼, 토글, 말풍선(부모=블루·아이=크림·AI=화이트)
- 녹음 위젯(파형+타이머), 빈 상태 일러스트(마스코트)
- **동화 플레이어**(페이지·읽기 하이라이트·코칭 말풍선·PEER 토글)
- **끝말잇기 체인 보드**(말풍선 체인·끝글자 강조·그림 힌트), **어휘 노트 카드**

## 8. 애니메이션 · 인터랙션 *(반드시 풍부하게)*
- **스플래시**: 로고 스케일 인 + 통통 bounce, 살짝 페이드
- **온보딩**: 슬라이드 전환 시 일러스트 패럴랙스, 둥둥 떠다니는 floating loop
- **홈 진입**: 카드들이 아래에서 위로 **순차(stagger) fade-up**
- **소통 점수 게이지**: 0→점수까지 원호가 채워지고 숫자 **카운트업**
- **차트**: 막대는 아래에서 자라나고(grow), 라인은 왼→오 **draw**, 도넛은 회전하며 채워짐
- **녹음**: 마이크 둘레 ripple 펄스 + 실시간 waveform, 녹음 시작/종료 시 버튼 morph
- **AI 분석 로딩**: 단계 체크가 하나씩 ✓로 바뀌고 점 로딩 맥동
- **동화 읽어주기**: 페이지 넘김(책장 넘기는 transition), 읽는 **단어 하이라이트가 따라가기**, 코칭 말풍선 pop-in, 마스코트 입모양 살짝 움직임
- **끝말잇기**: 새 낱말 말풍선이 체인으로 톡 이어지며 등장, 끝 글자 **강조 pulse**, 정답·이어가기 성공 시 스티커/컨페티 pop
- **버튼**: 누르면 scale 0.96 + 살짝 그림자 축소(햅틱 느낌)
- **탭 전환**: 콘텐츠 cross-fade, 활성 탭 아이콘 bounce
- **중앙 FAB**: 탭 시 3개 옵션이 **라디얼로 펼쳐짐**(스프링)
- **카드 → 상세**: shared element transition(카드가 확장되며 리포트/동화로)
- **Pull-to-refresh**: 마스코트가 통통 튀는 커스텀 인디케이터
- 전체 전환은 부드러운 ease-out, 240~320ms 권장

## 10. 코칭 콘텐츠의 교육학 근거 (화면에 명시할 것)
아래 원리를 리포트·코칭·동화·끝말잇기 화면의 코칭 문구로 **구체적인 한국어 예시와 함께** 노출.

- **열린 질문 vs 폐쇄형 질문** — 폐쇄형("재밌었어?")은 예/아니오로 대화가 끝나지만, 열린 질문("오늘 뭐가 제일 재밌었어?", "왜 그렇게 생각해?")은 더 길고 풍부한 말과 어휘를 끌어냄. → 리포트 「이렇게 말해보세요」, 코칭 레슨, 홈 코칭 카드에 Before→After로 노출.
- **대화식 책읽기 PEER** — 질문하기(Prompt) → 아이 답 인정(Evaluate) → 한 마디 더해 확장(Expand) → 새 낱말 반복(Repeat). → 동화 화면 PEER 가이드.
- **CROWD 질문 5종** — 완성형 / 회상 / 열린질문 / 육하원칙(누가·왜·어떻게) / 연결(아이 삶과 잇기). → 동화 화면 페이지별 질문 말풍선.
- **확장·되돌려주기(expansion·recast)** — 아이 "멍멍 가" → "응, 강아지가 집으로 가는구나"처럼 바르고 길게 되돌려주기. → 코칭 카드·리포트 제안.
- **충분히 기다려주기** — 질문 뒤 5초쯤 기다려 아이가 스스로 말하게. → 코칭 팁.
- **주고받기(서브 앤 리턴)** — 아이의 말·표정에 즉시 반응해 대화를 주고받기. → 홈·리포트 코칭 문구.
- **끝말잇기 = 음운인식** — 낱말의 끝소리에 주목하게 해 소리 단위(음소·음절) 인식과 어휘를 키움. 새 낱말은 어휘 노트로 누적. → 끝말잇기 화면 팁.
